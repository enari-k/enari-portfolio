# enari-portfolio

enari_K の活動、制作物、技術スタックを統合したポートフォリオサイトのソースコードです。

[![Live Site](https://img.shields.io/badge/Live_Demo-enari--k.com-3EA8FF?style=for-the-badge&logo=google-chrome&logoColor=white)](https://enari-k.com)

## 📌 コンセプト
「材料工学（物質理工）」と「情報工学（traP）」という、自身の二つのアイデンティティが交差する地点を表現するためのハブとして構築しています。
Zenn、AtCoder、Unityroom、Steam、SlideShare など、各プラットフォームに分散した活動を一箇所に集約しています。

## 🛠 技術スタック
モダンなフロントエンド技術を採用し、高速でメンテナンス性の高い構成を目指しています。

- **Framework:** Next.js (App Router)
- **Library:** React
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## ✨ 特徴
- **Responsive Design:** PC / スマートフォン両対応。
- **Activity Integration:** Zenn の記事一覧や GitHub の活動状況を可視化。
- **Game Portfolio:** 開発したゲームの紹介と、各配信プラットフォームへの導線。
